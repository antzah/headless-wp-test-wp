# Headless WordPress test site

This is the WP site that works as the backend for the Next.js frontend.

This is a practice project that follows the tutorial at [https://dev.to/kendalmintcode/configuring-wordpress-as-a-headless-cms-with-next-js-3p1o](https://dev.to/kendalmintcode/configuring-wordpress-as-a-headless-cms-with-next-js-3p1o)  

## Installation

* Update wp-config.php with your own development database credentials
* Populate your database with dummy content with a plugin like [Fakerpress](https://wordpress.org/plugins/fakerpress/)
